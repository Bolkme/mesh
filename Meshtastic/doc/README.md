## 什么是 Meshtastic

Meshtastic 是一个开源的、基于 LoRa 的 Mesh 网络项目，允许设备在没有蜂窝网络或互联网的情况下进行通信。

**特点：**
- 🌐 去中心化 Mesh 网络
- 📻 使用 LoRa 无线电技术
- 🔓 开源硬件和软件
- 💰 低成本
- 🔋 低功耗

---

## 核心概念

### LoRa (Long Range)
- 远距离无线通信技术
- 低功耗，适合电池供电
- 传输距离：城市 1-5km，开阔地 10km+

### Mesh 网络
- 每个节点都可以中继消息
- 自动路由，无需基础设施
- 网络自愈能力强

### 频道 (Channel)
- 用于隔离不同的通信组
- 支持加密
