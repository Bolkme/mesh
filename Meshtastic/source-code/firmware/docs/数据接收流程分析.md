# Meshtastic LoRa 数据接收流程分析

## 📊 数据流向总览

```mermaid
flowchart TD
    subgraph Physical_Layer["物理层 - RadioLibInterface"]
        A[LoRa 天线接收信号] --> B[SX126x/SX128x 芯片]
        B --> C[硬件中断 ISR_RX]
        C --> D[isrRxLevel0 中断处理]
        D --> E[onNotify ISR_RX]
        E --> F[handleReceiveInterrupt]
        F --> G[读取 radioBuffer]
        G --> H[构建 MeshPacket]
        H --> I[deliverToReceiver]
    end

    subgraph Router_Layer["路由层 - Router"]
        I --> J[enqueueReceivedMessage]
        J --> K[fromRadioQueue 队列]
        K --> L[Router::runOnce]
        L --> M[perhapsHandleReceived]
        M --> N{过滤检查}
        N -->|忽略列表 | O[丢弃包]
        N -->|通过 | P[handleReceived]
    end

    subgraph Decode_Layer["解码层"]
        P --> Q[perhapsDecode]
        Q --> R{解密方式}
        R -->|PKI 加密 | S[decryptCurve25519]
        R -->|PSK 加密 | T[channels.decryptForHash]
        R -->|已解码 | U[跳过解密]
        S --> V[pb_decode 解析 protobuf]
        T --> V
        U --> V
    end

    subgraph Module_Layer["模块层 - MeshModule"]
        V --> W[callModules]
        W --> X[遍历所有模块]

        Y -->|否 | AA[跳过]
        Z --> AB[alterReceived]
        AB --> AC{want_response?}
        AC -->|是 | AD[sendResponse]
        AC -->|否 | AE[继续下一个模块]
    end

    subgraph Service_Layer["服务层 - MeshService"]
        AD --> AF[sendToMesh]
        AE --> AF
        AF --> AG[toPhoneQueue]
        AG --> AH[手机/客户端]
        AF --> AI[MQTT 转发]
        AF --> AJ[路由转发]
    end

    Physical_Layer --> Router_Layer
    Router_Layer --> Decode_Layer
    Decode_Layer --> Module_Layer
    Module_Layer --> Service_Layer
```

## 🔄 接收中断处理详细流程

```mermaid
sequenceDiagram
    participant LoRa as LoRa 芯片
    participant ISR as 中断服务程序
    participant RadioIf as RadioLibInterface
    participant Router as Router
    participant Modules as MeshModule
    participant Service as MeshService

    LoRa->>ISR: 硬件中断 (RX_DONE)
    ISR->>RadioIf: isrRxLevel0()
    RadioIf->>RadioIf: disableInterrupt()
    RadioIf->>RadioIf: notifyFromISR(ISR_RX)
    
    Note over RadioIf: 线程上下文切换
    
    RadioIf->>RadioIf: onNotify(ISR_RX)
    RadioIf->>RadioIf: handleReceiveInterrupt()
    RadioIf->>LoRa: readData(radioBuffer)
    RadioIf->>RadioIf: 构建 MeshPacket
    RadioIf->>RadioIf: addReceiveMetadata<br/>(SNR/RSSI/时间戳)
    RadioIf->>Router: deliverToReceiver(mp)
    
    Router->>Router: enqueueReceivedMessage()
    Router->>Router: fromRadioQueue.enqueue()
    Router->>Router: setReceivedMessage()
    
    Note over Router: Router 线程唤醒
    
    Router->>Router: runOnce()
    Router->>Router: fromRadioQueue.dequeue()
    Router->>Router: perhapsHandleReceived()
    
    alt 包被过滤
        Router->>Router: packetPool.release()
    else 包通过过滤
        Router->>Router: handleReceived()
        Router->>Router: perhapsDecode()
        
        alt PKI 加密
            Router->>Router: decryptCurve25519()
        else PSK 加密
            Router->>Router: channels.decryptForHash()
        end
        
        Router->>Router: pb_decode_from_bytes()
        Router->>Modules: callModules()
        
        loop 遍历所有模块
            Modules->>Modules: wantPacket()
            alt 感兴趣
                Modules->>Modules: handleReceived()
                Modules->>Modules: alterReceived()
                alt 需要响应
                    Modules->>Modules: sendResponse()
                end
            end
        end
        
        Router->>Service: sendToMesh()
        Service->>Service: toPhoneQueue.enqueue()
    end
```

## 📦 关键数据结构

```mermaid

        +NodeNum to

    }

    class RadioBuffer {
        +PacketHeader header

    }

    class meshtastic_MeshPacket {
        +NodeNum from
        +NodeNum to
        +PacketId id
        +uint32_t rx_time
        +float rx_snr
        +int32_t rx_rssi
        +uint8_t hop_limit
        +uint8_t hop_start
        +bool want_ack
        +bool via_mqtt
        +ChannelIndex channel
        +union payload_variant
    }

    class meshtastic_Data {
        +PortNum portnum
        +bytes payload
        +bool want_response
        +uint32_t request_id
    }

    class Router {
        -RadioInterface iface
        -PointerQueue fromRadioQueue
        +enqueueReceivedMessage()
        +handleReceived()
        +send()
    }

    class RadioLibInterface {
        -Module module
        -PointerQueue txQueue
        +isrRxLevel0()
        +handleReceiveInterrupt()
        +send()
    }

    PacketHeader -- RadioBuffer : 包含
    meshtastic_MeshPacket -- meshtastic_Data : 解码后
    RadioLibInterface -- Router : 传递包
    RadioBuffer -- PacketHeader : 包含
```

## 📍 关键代码位置

| 功能 | 文件 | 关键函数 |
|------|------|----------|
| 中断处理 | `RadioLibInterface.cpp` | `isrRxLevel0()`, `handleReceiveInterrupt()` |
| 包队列 | `Router.cpp` | `enqueueReceivedMessage()`, `runOnce()` |
| 包过滤 | `Router.cpp` | `perhapsHandleReceived()` |
| 解密解码 | `Router.cpp` | `perhapsDecode()` |
| 模块分发 | `MeshModule.cpp` | `callModules()` |
| 服务分发 | `MeshService.cpp` | `handleFromRadio()`, `sendToPhone()` |

## 🔐 安全处理流程

```mermaid
flowchart LR
    A[接收加密包] --> B{解密方式}
    B -->|PKI 加密 | C[检查目标节点公钥]
    C --> D[Curve25519 解密]
    B -->|PSK 加密 | E[遍历所有信道]
    E --> F[尝试每个信道的 PSK]
    F --> G[AES-256 解密]
    D --> H{protobuf 解析}
    G --> H
    H -->|成功 | I[更新 channel 为索引]
    H -->|失败 | J[DECODE_FAILURE]
    I --> K[callModules 处理]
    J --> L[丢弃或记录]
```

## 📈 性能优化点

1. **中断快速处理**: ISR 只做最小工作，通知线程处理
2. **零拷贝设计**: 使用内存池分配，避免动态分配
3. **早丢弃策略**: 在解码前过滤无效包
4. **模块链式处理**: 支持模块提前终止处理链
5. **延迟重播**: 基于 SNR 的动态退避机制
