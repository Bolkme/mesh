## 🔌 物理连接方式

### 1. 蓝牙 BLE (低功耗蓝牙)

**GATT 服务 UUID**: `0xfe34` (Meshtastic 自定义服务)

**特征值 (Characteristics)**:
| UUID | 属性 | 用途 |
|------|------|------|
| `0x2f0c` | WRITE | ToRadio (手机 → 设备) |
| `0x2f0a` | NOTIFY | FromRadio (设备 → 手机) |

**连接流程**:
```
1. 扫描 BLE 设备 (名称：Meshtastic_XXXX)
2. 连接到 0xfe34 服务
3. 订阅 FromRadio 特征 (0x2f0a) 的 NOTIFY
4. 发送 ToRadio 配置请求到 0x2f0c
5. 接收 FromRadio 响应通知
```

### 2. WiFi (TCP/UDP)

**默认端口**:
- TCP: `4403`
- UDP: `4403`

**连接示例** (Python):
```python
import socket

# TCP 连接
sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
sock.connect(('192.168.4.1', 4403))

# 发送 ToRadio protobuf
toradio_bytes = to_radio.SerializeToString()
sock.sendall(len(toradio_bytes).to_bytes(2, 'big') + toradio_bytes)

# 接收 FromRadio protobuf
length = int.from_bytes(sock.recv(2), 'big')
fromradio_bytes = sock.recv(length)
```

### 3. 串口 (Serial/USB)

**波特率**: 115200 (默认)

**帧格式**:
```
┌──────────┬─────────────┬──────────┬──────────┐
│ 0x94     │ 长度 (2 字节) │ Protobuf  │ 0x93     │
│ 起始符   │ 大端序       │ 数据      │ 结束符   │
└──────────┴─────────────┴──────────┴──────────┘
```

**Python 串口示例**:
```python
import serial

ser = serial.Serial('/dev/ttyUSB0', 115200, timeout=1)

# 发送 ToRadio
toradio_bytes = to_radio.SerializeToString()
frame = b'\x94' + len(toradio_bytes).to_bytes(2, 'big') + toradio_bytes + b'\x93'
ser.write(frame)

# 接收 FromRadio
data = ser.read(3)  # 读取起始符 + 长度
if data[0] == 0x94:
    length = int.from_bytes(data[1:3], 'big')
    protobuf_data = ser.read(length + 1)  # +1 为结束符
```

## 📦 Protobuf 数据结构

### ToRadio (手机 → 设备)

```protobuf
message ToRadio {
  oneof payload_variant {
    MeshPacket packet = 1;      // 发送消息到 mesh
    RadioConfig config = 2;     // 配置无线电 (已废弃)
    User user = 3;              // 设置用户信息
    Position position = 4;      // 设置位置
    Config config_v2 = 5;       // 设备配置
    ModuleConfig module_config = 6;  // 模块配置
    bytes mqtt_client_proxy_message = 7;  // MQTT 代理
    CoreDeviceUIEvent device_ui_event = 8; // UI 事件
    XModem xmodem = 9;          // 文件传输
    Contact contact = 10;       // 联系人
    LocalConfig local_config = 11;   // 本地配置
    LocalModuleConfig local_module_config = 12;  // 本地模块配置
    bool disconnect = 14;       // 断开连接
    Heartbeat heartbeat = 15;   // 心跳包
  }
}
```

### FromRadio (设备 → 手机)

```protobuf
message FromRadio {
  uint32 id = 1;                // 包 ID (递增)
  
  oneof payload_variant {
    MyNodeInfo my_info = 2;     // 我的节点信息
    RadioConfig config = 3;     // 无线电配置 (已废弃)
    NodeInfo node_info = 4;     // 节点信息
    Packet packet = 5;          // 接收到的消息
    Config config_v2 = 6;       // 设备配置
    ModuleConfig module_config = 7;  // 模块配置
    Channel channel = 8;        // 信道配置
    uint32 complete_number = 9; // 完成计数
    LogRecord log_record = 10;  // 日志记录
    XModem xmodem = 11;         // 文件传输
    DeviceMetadata metadata = 12; // 设备元数据
    QueueStatus queue_status = 13; // 队列状态
    MqttClientProxyMessage mqtt_client_proxy_message = 14; // MQTT 代理
    Contact contact = 15;       // 联系人
    FileInfo file_info = 16;    // 文件信息
    ClientNotification notification = 17; // 客户端通知
  }
}
```

### MeshPacket (消息包)

```protobuf
message MeshPacket {
  NodeNum from = 1;             // 发送方节点号
  NodeNum to = 2;               // 接收方节点号 (0=广播)
  uint32 id = 3;                // 包 ID (唯一)
  
  uint32 rx_time = 4;           // 接收时间戳
  float rx_snr = 5;             // 信噪比
  int32 rx_rssi = 6;            // 信号强度
  
  uint8_t hop_limit = 7;        // 跳数限制
  bool want_ack = 8;            // 需要确认
  Priority priority = 9;        // 优先级
  
  uint32 rx_delay = 10;         // 接收延迟
  ChannelIndex channel = 11;    // 信道索引
  
  oneof payload_variant {
    Data decoded = 12;          // 解码后的数据
    bytes encrypted = 13;       // 加密数据
  }
  
  // ... 其他字段
}
```

### Data (应用数据)

```protobuf
message Data {
  PortNum portnum = 1;          // 端口号 (应用类型)
  bytes payload = 2;            //  payload 数据
  uint32 request_id = 3;        // 请求 ID (响应时使用)
  bool want_response = 4;       // 需要响应
  uint32 source = 5;            // 源地址
  uint32 dest = 6;              // 目标地址
}
```

## 🔄 通信流程

### 1. 初始化连接

```
手机                              设备
 │                                 │
 ├─── ToRadio: connect request ──> │
 │                                 │
 │                                 ├─── 发送 STATE_SEND_MY_INFO
 │                                 ├─── 发送 STATE_SEND_OWN_NODEINFO
 │                                 ├─── 发送 STATE_SEND_METADATA
 │                                 ├─── 发送 STATE_SEND_CHANNELS
 │                                 ├─── 发送 STATE_SEND_CONFIG
 │                                 └─── 发送 STATE_SEND_MODULECONFIG
 │                                 │
 │<── FromRadio: MyNodeInfo ──────┤
 │<── FromRadio: NodeInfo ────────┤
 │<── FromRadio: DeviceMetadata ──┤
 │<── FromRadio: Channel × 8 ─────┤
 │<── FromRadio: Config × N ──────┤
 │<── FromRadio: ModuleConfig × N ┤
 │                                 │
 └─── ToRadio: config complete ──> │
                                   │
                              进入 STATE_SEND_PACKETS 状态
```

### 2. 发送文本消息

```python
# Python 示例
from meshtastic.protobuf import mesh_pb2, portnums_pb2

# 创建 MeshPacket
packet = mesh_pb2.MeshPacket()
packet.to = 0  # 广播
packet.decoded.portnum = portnums_pb2.PortNum.TEXT_MESSAGE_APP
packet.decoded.payload = b"Hello Meshtastic!"

# 包装到 ToRadio
to_radio = mesh_pb2.ToRadio()
to_radio.packet.CopyFrom(packet)

# 发送...
```

### 3. 接收消息

```python
# 接收 FromRadio
from_radio = mesh_pb2.FromRadio()
from_radio.ParseFromString(data)

if from_radio.HasField('packet'):
    packet = from_radio.packet
    if packet.decoded.portnum == portnums_pb2.PortNum.TEXT_MESSAGE_APP:
        message = packet.decoded.payload.decode('utf-8')
        print(f"收到消息：{message}")
```

### 4. 订阅位置信息

```python
# 订阅位置更新
to_radio = mesh_pb2.ToRadio()
to_radio.want_config_id = True

# 设备会定期发送 POSITION_APP 的 FromRadio 包
# 解析位置数据
position = mesh_pb2.Position()
position.ParseFromString(packet.decoded.payload)

latitude = position.latitude_i * 1e-7
longitude = position.longitude_i * 1e-7
```

## 🔐 安全机制

### 加密方式

1. **PKI 加密** (点对点)
   - Curve25519 密钥交换
   - AES-GCM 加密
   - 需要双方公钥

2. **PSK 加密** (信道)
   - AES-256-CTR
   - 共享预共享密钥
   - 基于信道索引

### 认证流程

```
1. 设备生成 config_nonce (随机数)
2. 手机请求配置时携带 nonce
3. 设备验证 nonce 匹配
4. 发送配置数据
5. 手机确认配置完成
```
