## 📁 项目结构

```
mesh/
├── doc/                      # 📚 学习笔记和文档
│   ├── README.md             # Meshtastic 基础知识
│   ├── 总体学习报告.md        # 完整学习报告
```

## 🎯 项目目标

1. 学习 Meshtastic 网络原理
2. 搭建和配置 Mesh 节点
3. 测试不同场景下的通信距离
4. 探索实际应用场景
