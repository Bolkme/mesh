## 🎯 项目目标

1. 学习 Meshtastic 网络原理
2. 学习 MeshCore 架构设计
3. 对比分析两种方案
4. 搭建和配置 Mesh 节点
5. 测试不同场景下的通信距离
6. 探索实际应用场景

## 📚 学习资源

### Meshtastic

- **定位**: 大众化离网通信平台
- **特点**: 功能丰富、社区活跃、即插即用
- **文档**: `Meshtastic/doc/` (5 份，已编号)
- **推荐阅读**: `Meshtastic/doc/00-README.md`

### MeshCore

- **定位**: 嵌入式轻量级路由协议
- **特点**: 简洁高效、MIT 许可、开发者友好
- **文档**: `MeshCore/doc/` (9 份，已编号)
- **推荐阅读**: `MeshCore/doc/01-索引与资源.md`

## 🆚 快速对比

| 维度 | Meshtastic | MeshCore |
|------|------------|----------|
| 定位 | 终端用户 | 嵌入式开发者 |
| 许可 | GPL-3.0 | MIT |
| 代码量 | ~50000+ 行 | ~2000 行核心 |
| 文档数量 | 5 份 | 9 份 |
| 内存管理 | 动态分配 | 静态池 |
| 社区 | 大型 | 成长中 |
| 推荐场景 | 个人使用/hobby | 商业/定制开发 |

详细对比见 `MeshCore/doc/05-与 Meshtastic 对比.md`

## 🚀 快速开始

### Meshtastic

1. 阅读 `Meshtastic/doc/00-README.md` 开始
2. 阅读 `Meshtastic/doc/01-总体学习报告.md` 了解基础
3. 准备硬件设备（T-Beam/Heltec 等）
4. 刷写固件并配置
5. 开始实验！

### MeshCore

1. 阅读 `MeshCore/doc/01-索引与资源.md` 开始
2. 访问 https://flasher.meshcore.co.uk 刷写固件
3. 使用客户端应用连接

### 开发者

**Meshtastic**:
- 查看 `Meshtastic/doc/02-固件学习报告.md`
- 参考官方文档 https://meshtastic.org/docs

**MeshCore**:
- 查看 `MeshCore/source-code/meshcore/` 源码
- 阅读 `MeshCore/doc/08-固件源码解析.md`
- 参考 `MeshCore/doc/09-协议详解.md`
